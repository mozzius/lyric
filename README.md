### Tech

 - Python 3
 - Flask
 - Mongodb
 - Socket.io
 - SASS
 - jQuery

### Running this project

1. Install python modules (pip -r requirements.txt)
2. Set up `libsass`, the error message will tell you what to do
3. `py app/main.py`
